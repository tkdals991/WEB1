WEB이란 무엇인가?

처음 인터넷이 들어오면서 Web 1.0 시대가 시작이 되었는데요 이때는 그저 사용하는 사람들이 신문이나 방송같이 일방적으로 보내주는 정보를 받는 형태였구요 Web 2.0 시대가 되면서 일방적인 것에서 벗어나 이제는 함께 참여도 하고 공유도 하면서 많은 사람들과 교류하고 같이 생각을 할수가 있었지요. 이제는 Web 3.0의 시대가 시작이 되었는데요 과연 이 Web 3.0 은 어떤 세상일까요? 바로 Web 3.0 은 개인중심으로 모든것이 바뀌어가는 인터넷 세상을 의미한다고 하네요. 모든걸 독점하고 있던 커다란 기업들 중심의 방향이 개인 중심으로 바뀌면서 수익도 기업에서 개인에게도 돌아가는 형태로 바뀌는 시기라고 합니다.
